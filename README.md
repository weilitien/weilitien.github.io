# Wei-Li Tien — Personal Portfolio

Personal resume & portfolio website for Wei-Li Tien, Software Engineer in Test.

🌐 **Live site**: [weilitien.github.io](https://weilitien.github.io)

## Features

- Dark-themed, award-winning design inspired by high-end creative portfolios
- Custom animated cursor with ring-follow effect
- Scroll-triggered fade-up animations
- Sticky navbar with blur effect on scroll
- Tech stack ticker / marquee
- Fully responsive (mobile-friendly)
- No frameworks — pure HTML, CSS, JavaScript

## Deploy to GitHub Pages

1. Create a repo named `<your-username>.github.io`
2. Upload all files to the repo root
3. Go to **Settings → Pages → Source → Deploy from branch (main)**
4. Your site will be live at `https://<your-username>.github.io`

## Tech Stack

- HTML5 / CSS3 / Vanilla JS
- Google Fonts: DM Sans + DM Serif Display
- CSS custom properties for theming
- IntersectionObserver API for scroll animations
